COURSES AND DESCRIPTION

Data Analysis with Python
This course covers the basics of data analysis using Python. Topics include data manipulation, cleaning, and visualization using libraries like Pandas, NumPy, and Matplotlib. You will also learn to perform exploratory data analysis (EDA) and interpret results.
Html and Css
This course covers the basics of HTML and CSS, teaching you how to create and style web pages. You'll learn about HTML elements, attributes, and CSS selectors, properties, and values. By the end of the course, you'll be able to build simple web pages.
Intro to Cyber Security Specialization
This specialization introduces the fundamental concepts of cybersecurity. It covers cryptography, network security,and risk management. The course is designed for beginners and includes hands-on projects to help you understand how to protect systems and data from cyber threats.
